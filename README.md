# Todo app made in C

Run "gcc cjson/cJSON.c main.c -o main" in the app directory to compile the app before running

To run the app just type: "./main" in the terminal
### App Commands:

+ Add    : add --title='' --description=''
+ Remove : remove --title=''
+ Read   : read --title=''
+ List   : list
+ Help   : help
+ Exit   : exit


### Arguments:
+ --title or -t
+ --description or -d
